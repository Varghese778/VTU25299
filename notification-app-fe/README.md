# Stage 2 - Notifications React App

This is the frontend application built to consume and beautifully display the campus notifications.

### Tech Stack
- **React** with **Vite**
- **Material UI** for a clean, modern aesthetic
- **React Router** for seamless navigation between the Inbox and Priority tabs

### Setup & Run
1. Make sure you are in this directory.
2. Install the necessary packages (including our custom logging middleware):
   ```bash
   npm install
   ```
3. Start the dev server:
   ```bash
   npm run dev -- --port 3000
   ```
4. Open your browser to `http://localhost:3000` to see the app in action!
